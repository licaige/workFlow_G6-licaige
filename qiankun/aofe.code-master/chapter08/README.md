## API 规范

OpenAPI 规范，早先称为 Swagger 规范，它是一个用于描述、生成、使用和可视化 RESTful Web 服务的、机器可读接口文件的规范。其优点是：基于 OpenAPI 接口文件实现的应用程序，可以自动生成方法、参数和模型的文档。它有助于使文档、客户端库和源代码保持同步。

Dropbox - https://github.com/dropbox/dropbox-api-spec
