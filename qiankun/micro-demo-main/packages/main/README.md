# micro-app-ain

Vue 3 + Typescript + Vite + Pinia

> - 使用 [script-setup]([script setup docs](https://v3.vuejs.org/api/sfc-script-setup.html#sfc-script-setup))
>
> - 样式复写使用 `:deep`

## Running the app

```bash
# 安装依赖
$ pnpm i
# 启动 mock 服务，注意替换代理
$ pnpm run mock
# dev
$ pnpm run dev
# production
$ pnpm run build
```
