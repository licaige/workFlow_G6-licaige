## VUE 2

### Install via npm

```bash
npm install vue-datepicker-ui --save
```


### Install via yarn

```bash
yarn add vue-datepicker-ui
```

## VUE 3


### Install via npm

```bash
npm install vue-datepicker-ui@vue3 --save
```


### Install via yarn

```bash
yarn add vue-datepicker-ui@vue3
```
