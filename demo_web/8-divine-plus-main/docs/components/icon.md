# Icon

图标组件

## 组件 - 使用方式

- 方式一: 提供 `color-颜色` 和 `size-字体大小` 组件属性配置
- 方式二: 也可以通过 `class` 直接设置样式，通过 `!important` 提高权重

<Icon />

## 组件属性

| 属性  | 描述       | 类型          | 可选值 | 默认值  |
| ----- | ---------- | ------------- | ------ | ------- |
| color | 图标的颜色 | string        | -      | inherit |
| size  | 图标的大小 | string/number | -      | 16      |
