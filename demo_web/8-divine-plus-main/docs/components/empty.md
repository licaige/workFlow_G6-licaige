# Empty

空状态时的占位提示。

## 基本用法

<Empty/>

## 自定义图片

<EmptyCustom/>

## Empty 组件属性

| 属性        | 描述             | 类型   | 可选值 | 默认值 |
| ----------- | ---------------- | ------ | ------ | ------ |
| image       | 图片地址         | string | -      | -      |
| image-size  | 图片大小（宽度） | number | -      | -      |
| description | 文本描述         | string | -      | -      |

## 插槽

| 插槽名      | 说明           |
| ----------- | -------------- |
| default     | 自定义底部内容 |
| image       | 自定义图片     |
| description | 自定义描述文字 |
