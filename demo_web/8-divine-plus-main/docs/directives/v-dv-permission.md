# v-dv-permission 权限

## 使用案列

- 按钮 1 显示 - 有权限
- 按钮 2 不显示 - 无权限

<v-dv-permission />

## 指令配置项

| 指令值 | 值类型 | 具体                                                           |
| ------ | ------ | -------------------------------------------------------------- |
| value  | object | {target: [按钮显示的白名单数组], current:[当前用户的权限数组]} |
