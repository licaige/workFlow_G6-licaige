# 属性描述符

## 读取属性的描述符

```
Object.getOwnPropertyDescriptor(obj, 'a')
```

## 设置属性描述符

```
Object.defineProperty(obj, 'a', {
  value: 10,
  writable: false,
  enumerable: false, // for循环，Object.keys，直接打印obj都无法看到a属性
  get: function () {...},
  set: function (val) {...},
})
```

## 禁止属性增删改

```
Object.freeze(obj)
```

## 密封：禁止属性增加，可以修改

```
Object.seal(obj)
```
